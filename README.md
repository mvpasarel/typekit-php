Typekit Client implementation with PHP

## Install

Via Composer

``` bash
$ composer require mvpasarel/typekit-php
```

## Usage

``` php
$skeleton = new Mvpasarel\Typekit();
echo $skeleton->echoPhrase('Hello, Mvpasarel!');
```

## Testing

``` bash
$ phpunit
```
